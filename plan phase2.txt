│ Second Phase: Full-Sphere Cascading Extremes with Laplace Margins                                                            │
│                                                                                                                              │
│ Context                                                                                                                      │
│                                                                                                                              │
│ Phase 1 implements cascading extremes on the positive simplex using exponential margins and Dirichlet direction              │
│ distributions. The new mathematical theory extends this to full R^d with Laplace margins, L2 radial-angular decomposition on │
│  the unit sphere, von Mises-Fisher direction distributions, formal immigration-branching genealogy, Ogata thinning           │
│ simulation, and subcriticality constraints. This is a complete mathematical reformulation that captures both crashes AND     │
│ rallies with sign information.                                                                                               │
│                                                                                                                              │
│ No changes to existing web code. Everything lives in second_phase/ and is tested via a new Streamlit app.                    │
│                                                                                                                              │
│ ---                                                                                                                          │
│ Key Mathematical Changes: Phase 1 → Phase 2                                                                                  │
│ ┌─────────────────┬───────────────────────────────────┬──────────────────────────────────────────────┐                       │
│ │    Component    │              Phase 1              │                   Phase 2                    │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Margins         │ Exponential: Y = -log(1-F̂(z)) ≥ 0 │ Laplace: X = F_Lap^{-1}(F̂(z)) ∈ R            │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Norm            │ L1: R = Σ X_j                     │ L2: R = ‖X‖₂                                 │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Angular space   │ Positive simplex S^{d-1}_+        │ Full sphere S^{d-1}                          │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Direction dist  │ Mixture of Dirichlet              │ Mixture of von Mises-Fisher                  │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Features        │ ζ(W) = log(W) - mean(log(W))      │ ξ(W) = [W², W_iW_j] (spherical)              │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Hawkes kernel   │ k·exp(-Δ/τ)·φ(R)                  │ A·K(Δ)·κ(m,m')·φ(R) with attenuation         │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Genealogy       │ ψ/λ ratio only                    │ Explicit parent P_i, cluster C(k)            │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Simulation      │ Direct exponential sampling       │ Ogata thinning (exact PP simulation)         │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Constraints     │ None                              │ Subcriticality penalty (spectral radius < 1) │                       │
│ ├─────────────────┼───────────────────────────────────┼──────────────────────────────────────────────┤                       │
│ │ Token dim (d=3) │ 8 = 2×3 + 2                       │ 14 = 3 + 9 + 2                               │                       │
│ └─────────────────┴───────────────────────────────────┴──────────────────────────────────────────────┘                       │

│ Implementation Steps                                                                                                         │
│                                                                                                                              │
│ Step 1: Foundation — __init__.py, utils.py, preprocess.py                                                                    │
│                                                                                                                              │
│ utils.py: Re-export EmpiricalCDF, load_config, save_json, seed_all, ensure_dir from cascades.utils. Add                      │
│ normalize_to_sphere() and log_vmf_normalizing().                                                                             │
│                                                                                                                              │
│ preprocess.py: Reuse compute_log_returns, fit_garch from cascades.preprocess. New functions:                                 │
│ - laplace_quantile(u) → F_Lap^{-1}(u) = -sign(u-0.5)·log(1-2|u-0.5|)                                                         │
│ - laplace_cdf(x) → F_Lap(x) = 0.5 + 0.5·sign(x)·(1-exp(-|x|))                                                                │
│ - standardize_laplace(residuals, pit_clip) → X ∈ R^d, CDFs dict                                                              │
│                                                                                                                              │
│ Step 2: Data Pipeline — dataset.py, extremes.py                                                                              │
│                                                                                                                              │
│ dataset.py:                                                                                                                  │
│ - compute_radial_angular_l2(X) → R = ‖X‖₂, W = X/‖X‖₂ ∈ S^{d-1}                                                              │
│ - spherical_features(W) → [W, W², W_iW_j] giving d(d+3)/2 features (9 for d=3)                                               │
│ - build_token_sphere(w, r, dt) → [W, ξ(W), log R, log ΔT]                                                                    │
│ - build_events_sphere(X, timestamps, u_tau_fn) → events dict                                                                 │
│ - SphereEventSequenceDataset (same sliding-window pattern as Phase 1)                                                        │
│                                                                                                                              │
│ extremes.py:                                                                                                                 │
│ - SphericalQuantileMLP — input: [W, ξ(W)], output: u_τ(w) via Softplus                                                       │
│ - fit_sphere_threshold(W, R, tau) → (u_tau_fn, meta, net) using pinball loss                                                 │
│                                                                                                                              │
│ Step 3: Core Distributions — distributions.py                                                                                │
│                                                                                                                              │
│ - vMF: log_vmf_density(w, μ, κ, d) using analytic form for d=3: log C_3(κ) = log(κ) - log(4π) - log(sinh(κ))                 │
│ - vMF sampling: Wood's algorithm (1994) — rejection sampling on [-1,1] + rotation to align with μ                            │
│ - Mixture: sample_vmf_mixture(π, μ, κ) — sample component k then sample vMF(μ_k, κ_k)                                        │
│ - Truncated Gamma: same rejection approach as Phase 1                                                                        │
│ - Excitation kernel: A·exp(-Δ/τ)·κ(m,m')·φ(R) — vectorized computation                                                       │
│                                                                                                                              │
│ Step 4: Model — model.py                                                                                                     │
│                                                                                                                              │
│ SphericalCascadeTransformer(d_input, d_assets, cfg):                                                                         │
│                                                                                                                              │
│ - Encoder: Same causal Transformer as Phase 1 (input_proj → pos_emb → TransformerEncoder with causal mask)                   │
│ - Direction head (vMF mixture): π = softmax(π_head(h)), μ_k = normalize(μ_head_k(h)), κ_k = softplus(κ_head(h))              │
│ - Magnitude head: Same gauge_net pattern, truncGamma(a, β, u_τ)                                                              │
│ - Time head (Hawkes + attenuation): λ = μ + Σ_j k·exp(-Δ/τ)·κ(W_i,W_j)·φ(R_j)                                                │
│ - Attenuation net: sigmoid(MLP([W_i, W_j])) ∈ (0,1] — never zero, allows rare cross-group transmission                       │
│ - Subcriticality penalty: max(0, max_row_sum(kernel) - margin)²                                                              │
│ - log_likelihood() returns: log_p_w (vMF), log_p_r (truncGamma), log_p_t (PP: log λ - λ·ΔT), subcrit_penalty, kernel         │
│                                                                                                                              │
│ Step 5: Training — train.py, configs/phase2.yaml                                                                             │
│                                                                                                                              │
│ Full pipeline: download → GARCH → Laplace PIT → L2 decomposition → sphere threshold → build events → train                   │
│ SphericalCascadeTransformer.                                                                                                 │
│                                                                                                                              │
│ Loss: -(log_p_w + log_p_r + log_p_t).mean() + subcrit_weight · subcrit_penalty                                               │
│                                                                                                                              │
│ Artifacts saved to artifacts/phase2/: model.pt, quantile_model.pt, events.npz, cdfs.npz, meta.json                           │
│                                                                                                                              │
│ Step 6: Simulation + Genealogy — simulate.py, genealogy.py                                                                   │
│                                                                                                                              │
│ genealogy.py:                                                                                                                │
│ - compute_parent_probabilities(λ, ψ, kernel) → P(P_i=0) = μ_i/λ_i, P(P_i=j) = kernel[i,j]/λ_i                                │
│ - sample_parents(parent_probs) → stochastic parent assignment                                                                │
│ - extract_clusters(parents) → dict of immigrant → descendant list                                                            │
│ - Genealogy dataclass: parents, immigrant_mask, clusters, cascade_probs                                                      │
│                                                                                                                              │
│ simulate.py — Ogata thinning:                                                                                                │
│ 1. Compute intensity bound Λ* = safety_factor × Λ(T_last)                                                                    │
│ 2. Propose Δt ~ Exp(Λ*)                                                                                                      │
│ 3. Sample candidate mark (W, R) from vMF mixture + truncGamma                                                                │
│ 4. Accept with prob λ(t*,m*)/Λ*                                                                                              │
│ 5. On acceptance: sample parent P_{i+1}, append to history                                                                   │
│ 6. Stop when T > horizon or max_events reached                                                                               │
│                                                                                                                              │
│ Also: prompt_generate(asset_idx, magnitude, horizon, model, q_model) — constructs X_0 = value·e_{j0}, checks R_0 > u_τ(W_0), │
│  then runs Ogata thinning.                                                                                                   │
